[h1]World Clock Timezone Map Wallpaper[/h1]

Un reloj mundial para Wallpaper Engine que muestra la hora local, el estado de día o noche y el terminador solar en tiempo real sobre un mapa Mercator.

[img]https://raw.githubusercontent.com/GrayArashiAI/world-clock-timezone-map/main/assets/world-map-timezone-map-preview.gif[/img]

[h2]Funciones implementadas[/h2]

[list]
[*]Relojes mundiales y terminador solar en tiempo real
[*]Zonas horarias IANA y horario de verano
[*]Ciudad actual, ciudades predefinidas y ciudades JSON
[*]Mapas centrados en el Atlántico o el Pacífico
[*]Diez idiomas de interfaz
[/list]

[h2]GitHub[/h2]

El código fuente de este fondo está disponible como código abierto en GitHub: [url=https://github.com/GrayArashiAI/world-clock-timezone-map/]GitHub[/url]

[h2]Uso[/h2]

La lista de ciudades de IANA se usa para elegir las zonas horarias introducidas manualmente. La ciudad que se muestra no tiene que aparecer en esa lista; si no tiene una entrada IANA propia, elige una ciudad IANA de la misma zona horaria. Este proyecto utiliza la versión 2026b de IANA.

Lista de ciudades de IANA: [url=https://data.iana.org/time-zones/tzdb-2026b/zone.tab]lista de ciudades de IANA[/url]

[h3]Configurar mi ciudad[/h3]

Mi ciudad tiene tres valores opcionales; todos pueden dejarse en blanco:

[list]
[*][b]Mi zona horaria[/b]: introduce una zona horaria IANA, como [b]Asia/Shanghai[/b]; si se deja en blanco, se detecta automáticamente desde la zona horaria del sistema.
[*][b]Mi ciudad[/b]: introduce el nombre de ciudad que se mostrará; si se deja en blanco, se deduce automáticamente de la zona horaria.
[*][b]Mis coordenadas[/b]: introduce [b]latitud,longitud[/b], usando valores negativos para el sur y el oeste; si se deja en blanco, se deducen automáticamente de la zona horaria.
[/list]

IANA no tiene una entrada propia para Pekín. Como Pekín comparte la zona horaria [b]Asia/Shanghai[/b], usa estos valores para mostrar Pekín:

[list]
[*][b]Mi zona horaria[/b]: [b]Asia/Shanghai[/b]
[*][b]Mi ciudad[/b]: [b]Pekín[/b]
[*][b]Mis coordenadas[/b]: [b]39.9042,116.4074[/b]
[/list]

[h3]Añadir ciudades personalizadas[/h3]

“Más ciudades (ver descripción del Workshop)” admite uno o varios objetos de ciudad. Se pueden omitir los corchetes del array.

[h3]Especificar todos los datos[/h3]

Usando Bombay como ejemplo, especifica la zona horaria, el nombre y las coordenadas:

[code]
{"timeZone":"Asia/Kolkata","name":"Bombay","lat":19.076,"lon":72.8777}
[/code]

[h3]Especificar solo la zona horaria[/h3]

Usando Dublín como ejemplo, introduce solo su zona horaria IANA. El nombre y las coordenadas se completan automáticamente:

[code]
{"timeZone":"Europe/Dublin"}
[/code]

[b]timeZone[/b] es obligatorio; [b]name[/b], [b]lat[/b] y [b]lon[/b] son opcionales.

[h3]Añadir varias ciudades[/h3]

Separa los objetos de ciudad con comas:

[code]
{"timeZone": "Pacific/Honolulu"},
{"timeZone": "America/Anchorage"},
{"timeZone": "America/Los_Angeles"},
{"timeZone": "America/Mexico_City"},
{"timeZone": "America/New_York"},

{"timeZone": "Europe/London"},
{"timeZone": "Europe/Paris"},
{"timeZone": "Europe/Istanbul"},
{"timeZone": "Europe/Moscow"},
{"timeZone": "Asia/Dubai"},
{"timeZone":"Asia/Kolkata","name":"Bombay","lat":19.076,"lon":72.8777},
{"timeZone": "Asia/Bangkok"},
{"timeZone":"Asia/Shanghai","name":"Pekín","lat":39.9042,"lon":116.4074},

{"timeZone": "America/Sao_Paulo"},
{"timeZone": "America/Lima"},
{"timeZone": "America/Santiago"},

{"timeZone": "Africa/Johannesburg"},
{"timeZone": "Australia/Sydney"},
{"timeZone": "Pacific/Auckland"},
[/code]

[h1]World Clock Timezone Map Wallpaper[/h1]

Обои с мировыми часами для Wallpaper Engine, показывающие местное время городов, день и ночь и линию солнечного терминатора на карте мира в проекции Меркатора.

[img]https://raw.githubusercontent.com/GrayArashiAI/world-clock-timezone-map/main/assets/world-map-timezone-map-preview.gif[/img]

[h2]Реализованные функции[/h2]

[list]
[*]Мировые часы и линия дня и ночи в реальном времени
[*]Часовые пояса IANA и переход на летнее время
[*]Текущий, предустановленные и пользовательские JSON-города
[*]Карта с центром по Атлантическому или Тихому океану
[*]Десять языков интерфейса
[/list]

[h2]Использование[/h2]

Список городов IANA используется для выбора вводимых вручную часовых поясов. Отображаемый город не обязан быть в этом списке; если для него нет отдельной записи IANA, выберите город IANA в том же часовом поясе. В проекте используется версия IANA 2026b.

Список городов IANA: [url=https://data.iana.org/time-zones/tzdb-2026b/zone.tab]список городов IANA[/url]

[h3]Настройка своего города[/h3]

У своего города есть три необязательных значения; все их можно оставить пустыми:

[list]
[*][b]Мой часовой пояс[/b]: укажите часовой пояс IANA, например [b]Asia/Shanghai[/b]; если оставить пустым, он автоматически определяется по системному часовому поясу.
[*][b]Мой город[/b]: укажите название города для отображения; если оставить пустым, оно автоматически выводится из часового пояса.
[*][b]Мои координаты[/b]: укажите [b]широта,долгота[/b], для южной широты и западной долготы используйте отрицательные значения; если оставить пустым, они автоматически выводятся из часового пояса.
[/list]

В IANA нет отдельной записи для Пекина. Пекин использует часовой пояс [b]Asia/Shanghai[/b], поэтому для отображения Пекина укажите:

[list]
[*][b]Мой часовой пояс[/b]: [b]Asia/Shanghai[/b]
[*][b]Мой город[/b]: [b]Пекин[/b]
[*][b]Мои координаты[/b]: [b]39.9042,116.4074[/b]
[/list]

[h3]Добавление пользовательских городов[/h3]

Поле «Доп. города (см. описание в Мастерской)» принимает один или несколько объектов городов. Квадратные скобки массива можно опустить.

[h3]Указать все данные города[/h3]

На примере Мумбаи укажите часовой пояс, название и координаты:

[code]
{"timeZone":"Asia/Kolkata","name":"Мумбаи","lat":19.076,"lon":72.8777}
[/code]

[h3]Указать только часовой пояс[/h3]

На примере Дублина укажите только часовой пояс IANA. Название города и координаты будут добавлены автоматически:

[code]
{"timeZone":"Europe/Dublin"}
[/code]

[b]timeZone[/b] обязателен; [b]name[/b], [b]lat[/b] и [b]lon[/b] можно опустить.

[h3]Добавить несколько городов[/h3]

Разделяйте объекты городов запятыми:

[code]
{"timeZone": "Pacific/Honolulu"},
{"timeZone": "America/Anchorage"},
{"timeZone": "America/Los_Angeles"},
{"timeZone": "America/Mexico_City"},
{"timeZone": "America/New_York"},

{"timeZone": "Europe/London"},
{"timeZone": "Europe/Paris"},
{"timeZone": "Europe/Istanbul"},
{"timeZone": "Europe/Moscow"},
{"timeZone": "Asia/Dubai"},
{"timeZone":"Asia/Kolkata","name":"Мумбаи","lat":19.076,"lon":72.8777},
{"timeZone": "Asia/Bangkok"},
{"timeZone":"Asia/Shanghai","name":"Пекин","lat":39.9042,"lon":116.4074},

{"timeZone": "America/Sao_Paulo"},
{"timeZone": "America/Lima"},
{"timeZone": "America/Santiago"},

{"timeZone": "Africa/Johannesburg"},
{"timeZone": "Australia/Sydney"},
{"timeZone": "Pacific/Auckland"},
[/code]

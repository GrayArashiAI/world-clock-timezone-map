[h1]World Clock Timezone Map Wallpaper[/h1]

一款为 Wallpaper Engine 制作的世界时钟动态壁纸，在墨卡托世界地图上实时显示城市时间、昼夜状态与晨昏线。

[img]https://raw.githubusercontent.com/GrayArashiAI/world-clock-timezone-map/main/assets/world-map-timezone-map-preview.gif[/img]

[h2]实现功能[/h2]

[list]
[*]实时世界时钟与晨昏线
[*]IANA 时区与夏令时支持
[*]当前城市、预设城市与 JSON 自定义城市
[*]大西洋、太平洋两种地图布局
[*]10 种界面语言
[/list]

[h2]使用方法[/h2]

IANA 城市列表用于选择手动填写的时区。要显示的城市不必出现在列表中；如果没有对应条目，请选择同一时区的 IANA 城市。本项目使用的 IANA 版本为 2026b。

IANA 城市列表：[url=https://data.iana.org/time-zones/tzdb-2026b/zone.tab]IANA 城市列表[/url]

[h3]设置我的城市[/h3]

本地城市有三个可选值，全部都可以留空：

[list]
[*][b]我的时区[/b]: 填写 IANA 时区，例如 [b]Asia/Shanghai[/b]；留空时会从系统时区自动获取。
[*][b]我的城市[/b]: 填写要显示的城市名；留空时会根据时区自动推断。
[*][b]我的坐标[/b]: 填写 [b]纬度,经度[/b]，南纬和西经使用负数；留空时会根据时区自动推断。
[/list]

IANA 列表没有单独的北京条目，但北京与 [b]Asia/Shanghai[/b] 使用同一时区。要显示北京，可以填写：

[list]
[*][b]我的时区[/b]: [b]Asia/Shanghai[/b]
[*][b]我的城市[/b]: [b]北京[/b]
[*][b]我的坐标[/b]: [b]39.9042,116.4074[/b]
[/list]

[h3]添加自定义城市[/h3]

在”更多城市(见工坊说明)”中可以输入一个或多个城市对象。项目支持省略数组方括号。

[h3]完整填写城市信息[/h3]

以孟买为例，同时指定时区、名称和坐标：

[code]
{"timeZone":"Asia/Kolkata","name":"孟买","lat":19.076,"lon":72.8777}
[/code]

[h3]仅填写时区[/h3]

以都柏林为例，只填写 IANA 时区，城市名和坐标会自动补全：

[code]
{"timeZone":"Europe/Dublin"}
[/code]

[b]timeZone[/b] 必填；[b]name[/b]、[b]lat[/b] 和 [b]lon[/b] 均可省略。

[h3]同时添加多个城市[/h3]

多个城市对象直接用逗号分隔：

[code]
{"timeZone": "Pacific/Honolulu"},
{"timeZone": "America/Anchorage"},
{"timeZone": "America/Los_Angeles"},
{"timeZone": "America/Mexico_City"},
{"timeZone": "America/New_York"},

{"timeZone": "Europe/London"},
{"timeZone": "Europe/Paris"},
{"timeZone": "Europe/Istanbul"},
{"timeZone": "Europe/Moscow"},
{"timeZone": "Asia/Dubai"},
{"timeZone":"Asia/Kolkata","name":"孟买","lat":19.076,"lon":72.8777},
{"timeZone": "Asia/Bangkok"},
{"timeZone":"Asia/Shanghai","name":"北京","lat":39.9042,"lon":116.4074},

{"timeZone": "America/Sao_Paulo"},
{"timeZone": "America/Lima"},
{"timeZone": "America/Santiago"},

{"timeZone": "Africa/Johannesburg"},
{"timeZone": "Australia/Sydney"},
{"timeZone": "Pacific/Auckland"},
[/code]

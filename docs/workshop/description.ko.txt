[list]
[*]실시간 세계 시계와 주야 경계선

[*]현재 도시, 기본 도시, JSON 사용자 지정 도시
[*]대서양 중심 및 태평양 중심 지도

[h2]사용 방법[/h2]

IANA 도시 목록은 수동으로 입력하는 시간대를 고르는 기준입니다. 표시할 도시가 반드시 이 목록에 있을 필요는 없습니다. 별도 IANA 항목이 없으면 같은 시간대의 IANA 도시를 선택하세요. 이 프로젝트에서 사용하는 IANA 버전은 2026b입니다.

IANA 도시 목록: [url=https://data.iana.org/time-zones/tzdb-2026b/zone.tab]IANA 도시 목록[/url]

[h3]내 도시 설정[/h3]

내 도시에는 선택 입력값이 세 가지 있으며, 모두 비워 둘 수 있습니다:

[list]
[*][b]내 시간대[/b]: [b]Asia/Shanghai[/b] 같은 IANA 시간대를 입력합니다. 비워 두면 시스템 시간대에서 자동으로 가져옵니다.
[*][b]내 도시[/b]: 표시할 도시 이름을 입력합니다. 비워 두면 시간대에서 자동으로 추론합니다.
[*][b]내 좌표[/b]: [b]위도,경도[/b]를 입력합니다. 남위와 서경은 음수를 사용합니다. 비워 두면 시간대에서 자동으로 추론합니다.
[/list]

IANA에는 베이징 전용 항목이 없습니다. 베이징은 [b]Asia/Shanghai[/b]와 같은 시간대를 사용하므로, 베이징을 표시하려면 다음 값을 입력합니다:

[list]
[*][b]내 시간대[/b]: [b]Asia/Shanghai[/b]
[*][b]내 도시[/b]: [b]베이징[/b]
[*][b]내 좌표[/b]: [b]39.9042,116.4074[/b]
[/list]

[h3]사용자 지정 도시 추가[/h3]

“도시 추가(워크숍 설명 참조)”에는 하나 이상의 도시 객체를 입력할 수 있습니다. 배열 대괄호는 생략할 수 있습니다.

[h3]모든 도시 정보 지정[/h3]

뭄바이를 예로 들어 시간대, 이름, 좌표를 함께 지정합니다.

[code]
{"timeZone":"Asia/Kolkata","name":"뭄바이","lat":19.076,"lon":72.8777}
[/code]

[h3]시간대만 지정[/h3]

더블린을 예로 들어 IANA 시간대만 입력합니다. 도시 이름과 좌표는 자동으로 채워집니다.

[code]
{"timeZone":"Europe/Dublin"}
[/code]

[b]timeZone[/b]은 필수이며 [b]name[/b], [b]lat[/b], [b]lon[/b]은 생략할 수 있습니다.

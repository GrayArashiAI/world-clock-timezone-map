[h1]World Clock Timezone Map Wallpaper[/h1]

一款為 Wallpaper Engine 製作的世界時鐘動態桌布，在麥卡托世界地圖上即時顯示城市時間、晝夜狀態與晨昏線。

[img]https://raw.githubusercontent.com/GrayArashiAI/world-clock-timezone-map/main/assets/world-map-timezone-map-preview.gif[/img]

[h2]實現功能[/h2]

[list]
[*]即時世界時鐘與晨昏線
[*]支援 IANA 時區與日光節約時間
[*]目前城市、預設城市與 JSON 自訂城市
[*]大西洋、太平洋兩種地圖配置
[*]10 種介面語言
[/list]

[h2]使用方法[/h2]

IANA 城市列表用於選擇手動填寫的時區。要顯示的城市不必出現在列表中；如果沒有對應條目，請選擇同一時區的 IANA 城市。本專案使用的 IANA 版本為 2026b。

IANA 城市列表：[url=https://data.iana.org/time-zones/tzdb-2026b/zone.tab]IANA 城市列表[/url]

[h3]設定我的城市[/h3]

本地城市有三個可選值，全部都可以留空：

[list]
[*][b]我的時區[/b]: 填寫 IANA 時區，例如 [b]Asia/Shanghai[/b]；留空時會從系統時區自動取得。
[*][b]我的城市[/b]: 填寫要顯示的城市名稱；留空時會根據時區自動推斷。
[*][b]我的座標[/b]: 填寫 [b]緯度,經度[/b]，南緯和西經使用負數；留空時會根據時區自動推斷。
[/list]

IANA 列表沒有獨立的北京條目，但北京與 [b]Asia/Shanghai[/b] 使用同一時區。若要顯示北京，可以填寫：

[list]
[*][b]我的時區[/b]: [b]Asia/Shanghai[/b]
[*][b]我的城市[/b]: [b]北京[/b]
[*][b]我的座標[/b]: [b]39.9042,116.4074[/b]
[/list]

[h3]新增自訂城市[/h3]

在「更多城市(見工坊說明)」中可以輸入一個或多個城市物件。專案支援省略陣列方括號。

[h3]完整填寫城市資訊[/h3]

以孟買為例，同時指定時區、名稱和座標：

[code]
{"timeZone":"Asia/Kolkata","name":"孟買","lat":19.076,"lon":72.8777}
[/code]

[h3]僅填寫時區[/h3]

以都柏林為例，只填寫 IANA 時區，城市名稱和座標會自動補齊：

[code]
{"timeZone":"Europe/Dublin"}
[/code]

[b]timeZone[/b] 必填；[b]name[/b]、[b]lat[/b] 和 [b]lon[/b] 均可省略。

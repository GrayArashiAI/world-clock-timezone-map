# Third-Party Data

This project includes or derives generated assets from the following data sources:

- [IANA Time Zone Database](https://www.iana.org/time-zones), version `2026b`. The tz database code and data are in the public domain unless otherwise specified by its upstream license.
- [Unicode CLDR](https://cldr.unicode.org/), version `48.2.0`, under the Unicode License v3 (`Unicode-3.0`). The complete license text is stored at [`data/cldr/LICENSE`](data/cldr/LICENSE).
- [Natural Earth](https://www.naturalearthdata.com/), 1:110m land data. Natural Earth vector and raster map data are in the public domain.
